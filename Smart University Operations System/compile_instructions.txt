COMPILATION METHODS:
====================

METHOD 1: Using Makefile (Recommended)
---------------------------------------
# On Linux/Mac:
make all            # Build the project
make run            # Build and run
make clean          # Clean build files

# On Windows with MinGW:
mingw32-make all
mingw32-make run


METHOD 2: Manual Compilation (Windows)
---------------------------------------
# Using g++:
g++ -std=c++11 -Wall -Isrc src/main.cpp -o university_system.exe

# Run:
university_system.exe


METHOD 3: Manual Compilation (Linux/Mac)
-----------------------------------------
# Using g++:
g++ -std=c++11 -Wall -Isrc src/main.cpp -o university_system

# Run:
./university_system


METHOD 4: Using Code::Blocks IDE
---------------------------------
1. Create new project: Console Application (C++)
2. Add all .cpp files from src/ to project
3. Add src/ directory to include paths
4. Build and run (F9)


METHOD 5: Using Visual Studio Code
-----------------------------------
1. Open project folder
2. Install C/C++ extension
3. Press Ctrl+Shift+B to build using Makefile
4. Or use terminal: g++ -std=c++11 -Isrc src/main.cpp -o university_system


REQUIREMENTS:
=============
- C++11 compatible compiler (g++, clang++, MSVC)
- No external libraries required
- All data structures manually implemented


FEATURES IMPLEMENTED:
=====================

MODULE 1: Student Record Management
- Dynamic Array for storage
- Hash Table for O(1) search
- AVL Tree for CGPA ranking
- Full CRUD operations

MODULE 2: Course Registration System
- Singly Linked List implementation
- Dynamic course add/drop
- Efficient insertion/deletion

MODULE 3: Library Management System
- Hash Table for book lookup
- AVL Tree for sorted listing
- Queue for reservations

MODULE 4: Complaint & Token Management
- Queue for normal complaints
- Circular Queue for tokens
- Min Heap (Priority Queue) for emergencies

MODULE 5: Campus Navigation System
- Graph with Adjacency List
- BFS for shortest path (unweighted)
- Dijkstra for shortest path (weighted)

MODULE 6: Undo/Redo System
- Stack for undo operations
- Stack for redo operations
- LIFO behavior demonstration

MODULE 7: Recently Viewed Records
- Doubly Linked List
- Browser-like navigation
- Bidirectional traversal

MODULE 8: Search & Sort Analysis
- Linear Search
- Binary Search
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Performance comparison with runtime analysis

TROUBLESHOOTING:
================

Problem: Compilation errors
Solution: Ensure C++11 support (-std=c++11 flag)

Problem: "command not found"
Solution: Install g++ compiler or use IDE

Problem: File not found errors
Solution: Run from project root directory

Problem: Data not persisting
Solution: Check if data/ directory exists and is writable
